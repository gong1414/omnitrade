# 行为准则

> [English](CODE_OF_CONDUCT.md) · 简体中文

本项目逐字采用 [Contributor Covenant 2.1][cc] 行为准则。完整原文请
点链接阅读（项目侧没有再做 reword）。

## 上报

如果你经历或目睹了违反 Covenant 的行为，请通过 GitHub 仓库 profile
上列的邮箱私下联系维护者。所有上报会保密处理。维护者会在 72 小时内
确认，并和上报人后续沟通处理路径。

安全漏洞（不是人际行为）请走 [SECURITY_ZH.md](SECURITY_ZH.md)，那
是另一条独立的私有上报通道。

## 适用范围

Covenant 适用于本项目所有空间 —— issue、PR、Discussion、commit
信息，以及任何托管在仓库的沟通方式 —— 也适用于代表本项目对外的场合。

## 执行

维护者可以根据自由裁量编辑、删除或拒绝不符合 Covenant 的贡献，并对
严重或反复违反者临时或永久限制参与权。

[cc]: https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/
