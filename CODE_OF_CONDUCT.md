# Code of Conduct

> English · [简体中文](CODE_OF_CONDUCT_ZH.md)

This project adopts the [Contributor Covenant, version 2.1][cc] verbatim.
Read the full text at the link.

## Reporting

If you experience or witness behaviour that violates the Covenant, please
contact the maintainers privately at the email address listed on the
GitHub repository profile. Reports are handled confidentially. The
maintainers will acknowledge within 72 hours and follow up with the
reporter on the resolution path.

For security issues (not interpersonal conduct), see [SECURITY.md][sec]
instead — those use a separate private reporting channel.

## Scope

The Covenant applies to all project spaces — issues, pull requests,
discussions, commit messages, and any other repo-hosted communication —
as well as off-platform interactions when an individual is officially
representing the project.

## Enforcement

Maintainers may, at their discretion, edit, remove, or reject
contributions that don't follow the Covenant, and may temporarily or
permanently restrict participation for serious or repeated violations.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[sec]: SECURITY.md
